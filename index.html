Instructions

Set your width and height using the above controls and press reset to change the size.
A height of 0 means that it will be a square. Note you generally want your width to be greater than your height.
Pressing "shuffle" will (obviously) shuffle the tiles.

Use the arrow keys to move the selection cross hair.
Hold Shift and press an arrow key to move the tiles in the selection in that direction

Try to get the board looking like it originally did. The time starts when you make your first move.
